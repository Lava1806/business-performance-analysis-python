# Business Performance & Profitability Analysis

## 📌 Objective

To analyze retail sales data and identify key drivers of revenue and profitability, and provide strategic business recommendations.

## 🛠️ Tools Used

* Python
* pandas
* matplotlib
* seaborn

## 📊 Dataset

Retail superstore dataset containing transaction-level data including sales, profit, discount, category, region, and customer details.

## 🔍 Key Analysis Performed

* Category-wise sales and profit analysis
* Discount vs profitability analysis
* Regional performance analysis
* Customer-level revenue analysis

## 📈 Key Insights

* High discounting negatively impacts profitability
* Certain categories generate high sales but low profit
* Sales performance varies across regions
* Revenue is concentrated among a few key customers

## 💼 Business Recommendations

* Optimize discount strategies to improve margins
* Focus on high-profit product categories
* Improve performance in underperforming regions
* Strengthen relationships with high-value customers
